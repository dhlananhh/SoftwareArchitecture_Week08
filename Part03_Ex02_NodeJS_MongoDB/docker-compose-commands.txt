Chạy stack:
    docker compose up -d --build


Kiểm tra:
    ■ Kiểm tra logs của 2 service:
        docker compose logs app
    và
        docker compose logs mongodb.
    Chú ý thông báo kết nối thành công từ app.

    ■ Mở trình duyệt hoặc dùng curl:
        curl http://localhost:3000/
        -> Nên trả về "Node.js + MongoDB API is running!"
        Tạo item mới:
            curl -X POST http://localhost:3000/items -H "Content-Type: application/json" -d '{"name":"My Item", "quantity": 10}'
        Lấy danh sách items:
            curl http://localhost:3000/items


Clean up:
    docker compose down -v
